# Data-Driven Pricing Strategy Project

This project implements a structured, data-driven pricing framework for an
e-commerce catalog of eco-friendly tableware products.

## Objective
Design a systematic pricing strategy that balances profitability, inventory
health, demand, and market competition.

## Approach
- Data cleaning and aggregation across pricing, sales, inventory, returns, and ads datasets
- Cost-based price protection using minimum and target margins
- Rule-based pricing recommendations (Increase / Maintain / Investigate)
- SKU-level final price recommendations

## Outputs
- `final_pricing_base_table.csv`
- `final_pricing_with_recommendations.csv`

## Tools Used
- Python
- Pandas
